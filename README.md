Objetivo
=========
O objetivo deste jogo consiste em deslocar todos os discos da torre onde se encontram para uma torre diferente, normalmente a terceira, respeitando as seguintes regras:

1. Deslocar um disco de cada vez, o qual deverá ser o do topo de uma das três torres;
2. Um disco nunca poderá ser colocado sobre outro de tamanho menor.
